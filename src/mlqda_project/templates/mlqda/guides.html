Guides

You can find out more about the proposed steps for using the system on this page. Following these steps helps you to lower the chances of encountering an error.

For more information, please check out the FAQ and About pages below.

Step 1 - Prepare your files

Please follow the guidelines for preparing your files. It is crucial that you appropriately edit your files before uploading. Otherwise, the system can return faulty results. You should only include texts that you want the analyse.

Make sure you use the correct file extensions when using the site. Currently, the system accepts files with .txt, .pdf, .docx, .csv and .xlsx extensions. Remember that for topic modelling, if you ise.docx, .txt and .pdf files, you must upload two or more documents.

Step 2 - Select the analyser

You can select from two analysers currently. Using the sentiment analyser returns a sentiment score for each of your sentences or entries from your files. The system also aggregates these scores for a more generalised understanding.

You can also choose to analyse your qualitative data with topic modelling. This method allows you to explore underlying topics in your input. The system provides you with a variety of results in this case.

Step 3 - Run the analysis

Using the upload button, select the files you wish to analyse. Remember that for topic modelling if you use .docx, .txt and .pdf files, you must upload two or more documents as the script requires multiple to identify phrases appearing across your data set.

Clicking 'Get Results' will redirect you to the results section. This process might take a few minutes as the system tries to optimise some aspects to provide better results.

Step 4.1 - Check the results. (Sentiment Analysis)

In the case of sentiment analysis, your results are compiled into one PDF file. Sentiment scores are shown next to every sentence or entry. The system also aggregates these scores to provide an overview for you.

Step 4.2 - Check the results. (Topic Modelling)

If you choose a Topic Modelling analysis, the system compiles your results into a ZIP folder. Within this folder, you can find a .json file representing the most important words for each topic the system identified. These word and contribution pairs are also displayed on the result page. Furthermore, the system creates a plot to visually represent each word's contribution to each topic. Finally, for each identified topic, the system compiles a PDF file to highlight the sentences containing at least one related word. It is important to note that the topics are not named, so you need to manually deduct the meaning of each topic based on the most important words.

Step 5 - Download and delete the results

From the result page, you can download your results for later analysis. The system automatically deletes every file and reference to the input files for data protection. The only file that is kept on the server is the exact file available for download. However, you need to delete this file using the delete button manually. If you fail to do so, the system will automatically delete these after every hour.
